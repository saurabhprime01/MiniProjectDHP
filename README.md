# Telecom Customer Churn Analysis

## Project Overview

This mini project focuses on analyzing telecom customer churn data using Python libraries like NumPy and Pandas. The objective is to explore customer behavior, identify churn patterns, analyze revenue, and generate useful business insights through data analysis.

The project helps understand:

- Customer churn behavior
- Revenue by contract type
- Monthly charge patterns
- Customer retention trends
- High-value customers

---

# Technologies Used

- Python
- NumPy
- Pandas
- Jupyter Notebook
